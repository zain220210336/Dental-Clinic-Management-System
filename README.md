
# Dental Clinic Management System

A desktop application developed using **Python**, **Tkinter**, and **SQLite** to simplify the daily management of dental clinic operations.

## 📌 Features

- 👤 Patient Management
- 📅 Appointment Management
- 🦷 Dental Treatment Records
- 🩻 X-ray Management
- 🦷 Implant Records
- 💾 SQLite Database Integration
- 🖥️ User-Friendly Interface

---

## 🛠️ Technologies Used

- Python
- Tkinter
- SQLite
- Git & GitHub

## 🚀 How to Run

1. Clone or download this repository.
2. Install Python 3.x.
3. Install the required libraries.

```bash
pip install pillow
```

4. Run the application.

```bash
python Clinic.py
```
